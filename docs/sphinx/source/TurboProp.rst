TurboProp
=========
This section documents the ``TurboProp`` class encoded in the
``isentropic_models.py`` file.  This class contains a single 
function that can be used to model the fluid properties at
the exit to the propeller, diffuser, compressor, heat addition component,
turbine, and the nozzle within a ram jet engine.
